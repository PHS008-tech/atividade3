Principais Formatações

Negrito/Destaque

Nesta frase temos um termo em negrito para isso se utiliza a tag < b > e ela não tem semântica

Nesta frase, tenho um termo em destaque para isso se utiliza a tag < strong > e ela é semântica

Italico/Ênfase

Nesta frase temos um termo em Italico usando a tag < I > e ela não é semântica

Nesta frase temos um termo em Ênfase usado a tag < em > e ela é semântica

Texto Marcado

Podemos criar um texto marcado utilizando a tag < mark >

Texto Grande e Pequeno

Estamo criando uma palavra Grande e outra Pequeno para coloca ela grande se utiliza a tag <big> e para a pequena se utiliza a tag <small> (n recomendo usa essas tags, é mais facil utiliza a tag <background-size> ela determina o tamanha da fonte e vc pode escolher o tamanho, mais pra frente vai aprende como utiliza ela, ent n só relaxa)

Texto Deletado

Podemos criar uma palavra ou um texto "excluido", que pode-se ler, mas não será considerado, só utilizar a tag <del>

Texto Incluindo

Podemos criar uma palavra ou um texto "incluindo" para indicar que ele foi adicionado recentemente, só utiliza a tag <ins>

Texto Sobrescrito

Para inserir coisas texto pra cima como coisas tipo "X20+3" se utiliza a tag <sup>

Texto Subescrito

Para inserir textos pra baixo como coisas do tipo "H2O" se utiliza a tag <sub>
